# Wildberries market input

Файл `demiand_wb_market.csv` предназначен для рыночных данных Wildberries:

- цена;
- наличие;
- ссылка на карточку;
- рейтинг/отзывы, если есть;
- срок.

Технические характеристики, фото и описание товара сюда не переносятся. Они остаются из official-layer.

Пустой файл только с заголовком разрешён и не активирует товары.
