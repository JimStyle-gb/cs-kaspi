# Manual real market input

Файл `demiand_manual_market_real.csv` нужен для ручной боевой цены/наличия, если данные берутся не напрямую из Ozon/WB или временно проверяются вручную.

Не путать с тестовым `demiand_manual_market.csv`, который должен оставаться пустым после patch 11.

Если заполняешь ручной файл как боевой, обязательно указывай `url`, `price`, `available`, `stock`, `lead_time_days`.
